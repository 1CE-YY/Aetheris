# RAG 系统测试文档

这是一个用于测试资源入库功能的 Markdown 文档。

## 系统概述

本系统是一个基于 RAG（Retrieval-Augmented Generation）的学习资源检索与推荐系统。

## 核心功能

### 1. 资源入库
- 支持 PDF 文件上传
- 支持 Markdown 文件上传
- 自动文本提取和切片
- 向量化入库

### 2. 语义检索
- 基于向量相似度的语义搜索
- 返回相关度最高的学习资源
- 提供引用来源和证据片段

### 3. 智能推荐
- 根据用户查询历史推荐资源
- 基于用户画像个性化推荐
- 提供推荐理由和学习建议

## 技术架构

- **后端**: Spring Boot 3.5 + Java 21
- **数据库**: MySQL 8.0
- **向量存储**: Redis Stack
- **前端**: Vue 3 + TypeScript
- **UI 框架**: Ant Design Vue

## 使用场景

学生可以上传学习资料（PDF 讲义、Markdown 笔记），然后通过自然语言查询相关内容。

系统会返回准确的答案，并标明引用来源，方便学生追溯信息。
